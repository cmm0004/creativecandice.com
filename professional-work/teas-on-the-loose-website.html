Tea Website

Teas on the Loose is an opportunity for me to share my passion for the ritual of loose leaf tea with others. It's my start-up business, and is finally ready to launch. Currently pulling all the paperwork together. This is the storefront website.

Website


View Live Site

The website is one-pager built with the BootStrap framework, and the blog is a wordpress installation in separate directory, for security reasons. This way, if the wordpress becomes compromised, I can wipe it and upload a backup, without ever effecting the money-making part of the site.
