Aruvio Product UI

I was asked to provide guidance on the re-design of the UI of our main product.

I created a set of high-fidelity mock-ups based on screenshots of the (then) current UI, using Photoshop.

Because the product was already built, I was quite limited to any structural changes I thought should be made, however I made some styling tweaks and suggestions for enhancing the UX by reducing the number of pages a customer is required to click through.

The Start Screen

Or the About Screen, was the only screen that I was given a set of things that needed changing, specifically the way the text and logos were arranged on the left side of the screen.

The original:

My changes:

Changes made to the menu/ header area:

  1. I moved the menu links to the right side to consolidate the buttons. I think it makes more sense for the menu buttons and the log-out button to be on the same side of the page.
  2. Menu buttons have icons and are in a ‘pill’ format. Bootstrap has the css for pill buttons.
  3. Active, inactive, and hover states are in brand colors.
  4. logout/ username is higher on the page and include a ‘Welcome, username’ A small change, but it makes it seem more accessible.
  5. Removed blue bar and extended light blue header.
  6. Updated the logo to the new one with ‘a virima company’ at the bottom.

Changes made to the content area:

  1. Made text bigger.
  2. removed the frames. They aren’t necessary and they date the design.
  3. At least 5px padding around the video.
